# 📝 Blog App
- A simple blog application built with Node.js, Express, MongoDB, and EJS.  
Users can sign up, log in, and create blogs with cover photos.  
Future updates will include improved design, authentication features, and more.
- Withour Signup/Login a user can also see Blogs.

# 🚀 Features
- User signup and login with JWT authentication
- Create and view blogs
- Upload cover photos
- Display author name and creation date
- Dynamic navigation bar (shows login/signup or logout/add blog depending on auth state)

# 🛠 Tech Stack
- Backend: Node.js, Express.js
- Database: MongoDB + Mongoose
- Frontend: EJS (templating), CSS
- Authentication: JWT, Bcrypt
- File Uploads: Multer
